Complemento Subtitle Downloader para NVDA

Visão Geral

O Subtitle Downloader é um complemento para o leitor de telas NVDA que permite baixar legendas de vídeos de diversas plataformas online diretamente para o seu computador.

Funcionalidades

  • Baixa legendas de vídeos usando o atalho NVDA+Shift+L.
  • Funciona em diversas plataformas populares como YouTube, Vimeo, Coursera e outras suportadas pela biblioteca yt-dlp.
  • Salva as legendas automaticamente na sua pasta de Downloads.
  • Usa o título do vídeo como nome do arquivo de legenda (formato: TituloDoVideo.idioma.txt).
  • Se múltiplas legendas estiverem disponíveis, exibe uma lista para você escolher o idioma desejado.
  • Converte as legendas para o formato de texto simples (.txt) para fácil leitura.
  • Pronto para tradução (internacionalizado).

Instalação

  1. Baixe o arquivo do complemento (SubtitleDownloader-X.Y.nvda-addon).
  2. Abra o NVDA.
  3. Vá ao menu NVDA -> Ferramentas -> Gerenciar complementos.
  4. Clique no botão "Instalar...".
  5. Navegue até o arquivo .nvda-addon que você baixou e selecione-o.
  6. Confirme a instalação e reinicie o NVDA quando solicitado.

Como Usar

  1. Navegue até a página web que contém o vídeo do qual você deseja baixar a legenda (ex: YouTube, Vimeo).
  2. Certifique-se de que a janela do navegador ou o player de vídeo esteja em foco.
  3. Pressione o atalho NVDA+Shift+L.
  4. O complemento tentará detectar a URL do vídeo e buscar por legendas.
  5. Se apenas um idioma for encontrado: A legenda será baixada e salva automaticamente na pasta Downloads. Você ouvirá uma mensagem de confirmação.
  6. Se múltiplos idiomas forem encontrados: Uma caixa de diálogo aparecerá listando os idiomas disponíveis. Use as setas para cima/baixo para selecionar o idioma desejado e pressione Enter. A legenda selecionada será baixada e salva.
  7. Se nenhuma legenda for encontrada ou ocorrer um erro: Você ouvirá uma mensagem informando o problema.

Plataformas Suportadas

O complemento utiliza a biblioteca yt-dlp nos bastidores, que suporta centenas de sites. A funcionalidade é mais testada e garantida em plataformas como:

  • YouTube
  • Vimeo
  • Dailymotion
  • Coursera (pode exigir login)
  • E muitas outras...

A capacidade de baixar legendas depende da disponibilidade no site e do suporte contínuo do yt-dlp.

Nota Importante sobre Autenticação e Captchas

Algumas plataformas (como YouTube e Vimeo em certas situações) podem exigir que você esteja logado ou que resolva verificações de segurança (captchas) para acessar o conteúdo, incluindo legendas. O complemento, por si só, não consegue fazer login ou resolver captchas.

Se o download falhar com mensagens de erro relacionadas a login ou verificação, pode ser necessário:

  • Garantir que você está logado na plataforma no seu navegador.
  • Para usuários avançados: Configurar o yt-dlp para usar os cookies do seu navegador. Isso geralmente envolve exportar os cookies do navegador para um arquivo e configurar o yt-dlp (ou uma futura versão deste complemento) para usá-los. Consulte a documentação do yt-dlp sobre "cookies" para mais detalhes.

Testes em ambientes automatizados (como o sandbox onde este complemento foi desenvolvido) podem não refletir completamente o uso real devido a essas restrições.

Solução de Problemas

  • Nenhuma URL detectada: Certifique-se de que o foco do NVDA está na janela correta (navegador ou player). Tente focar a barra de endereços ou o próprio vídeo.
  • Erro de download: Verifique sua conexão com a internet. O vídeo pode não ter legendas ou a plataforma pode estar bloqueando o acesso (veja a nota sobre autenticação).
  • yt-dlp não encontrado: Esta dependência é crucial. Se o complemento foi empacotado corretamente, ela deve estar incluída. Se você instalou manualmente, certifique-se de que yt-dlp está acessível pelo Python do NVDA.
